# Exercise - Temperature Converter
- Write a program that, based on a temperature in degrees Celsius, converts it and displays it in Kelvin (K), Réaumur (Re), Rankine (Ra) and Fahrenheit (F), following the formulas:
  - F = C * 1.8 + 32;
  - K = C + 273.15;
  - Re = C * 0.8;
  - Ra = C * 1.8 + 32 + 459.67;
